# mingw-LearnC

　　这个项目主要是用于新手在mingw环境下学C。

　　在git上我也看到了有人放上这个Reading-Party/lchwex「笨办法学C（旧版）」。但该整理者应该是在linux平台下工作的，并不适合mingw.之后又在百度和google上抓了很多源码。整理后给新人参考。因为用到的所有源码都是在网络上公共资源中搜索出来不知道来源，如果有版权的问题，请和我联系删除。

　　各主程序源码基本不做改动，只是会加上一些初学者常用到的源码。尽可能多的源码说明，初学者应该喜欢更多的注释。在makefile部分，基本上已经可以入门够用~ 对linux和win平台的不同，文中大部分都有解决方案。想深入请百度~


使用说明：
　　对于单文件的源码编译，可以用主目录下的makefile和批处理直接编译。
